# SWE-Burnout-Classification
This project explores the relationship between developer work habits and productivity, with a focus on identifying early signs of burnout. Using a dataset of developer behavioral and environmental metrics, the project performs exploratory data analysis (EDA) and visualizations to uncover patterns that may impact performance.

The long-term goal is to support predictive modeling that can classify burnout levels and help organizations intervene early.

Objectives: 
- Analyze factors that influence developer productivity
- Explore potential indicators of burnout
- Visualize relationships between behavioral variables (e.g., caffeine intake, bugs per day)
- Lay groundwork for future predictive modeling

Dataset
The dataset used in this project contains developer-related metrics such as:
- Caffeine intake
- Bugs introduced per day
- Work patterns and behavioral indicators

Note: The dataset is loaded from Google Drive in the notebook.

Technologies Used: 
- Python
- Pandas
- Matplotlib
- Google Colab

Key Analysis:
- Data inspection and preprocessing
- Feature exploration
